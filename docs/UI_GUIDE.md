## 🖼️ 主窗口布局

当你运行 `./run.sh` 后，会看到一个 **1200x800** 的主窗口，包含以下区域：

### 1. 窗口标题栏
```
┌────────────────────────────────────────────────────────┐
│ VideoEye 2.0 - 视频流分析软件              [─][□][×] │
└────────────────────────────────────────────────────────┘
```

### 2. 菜单栏 (顶部)

#### 📁 文件菜单
- **打开文件** (`Ctrl+O`) - 打开本地视频文件
  - 支持格式: mp4, avi, mkv, flv, ts, mp3, aac, wav
- **打开URL** (`Ctrl+U`) - 打开网络流媒体
  - 支持协议: RTMP, RTSP, HTTP
- **退出** (`Ctrl+Q`) - 关闭程序

#### ▶️ 播放菜单
- **播放** (`Space`) - 开始播放
- **暂停** - 暂停播放
- **停止** (`Esc`) - 停止播放

#### 📊 分析菜单
- **流分析** - 分析码流信息（开发中）
- **人脸检测** - 实时人脸检测（开发中）
- **直方图分析** - 帧直方图分析（开发中）

#### ❓ 帮助菜单
- **关于** - 显示版本信息

### 3. 工具栏 (菜单栏下方)

快速访问按钮:

## 🖱️ 交互功能

### 当前可用功能

1. **打开文件**
   ```
   点击: 文件 → 打开文件
   快捷键: Ctrl+O
   功能: 选择本地视频文件并打开
   ```

2. **打开网络流**
   ```
   点击: 文件 → 打开URL
   快捷键: Ctrl+U
   功能: 输入 RTMP/RTSP/HTTP 地址
   默认值: rtmp://
   ```

3. **播放控制**
   ```
   播放 → 开始解码并显示视频
   暂停 → 暂停解码
   停止 → 停止解码并清空显示
   ```

4. **进度跳转**
   ```
   拖动进度条 → 跳转到指定位置
   实时显示当前时间和总时长
   ```

### 开发中功能

- ❌ 视频帧显示 (需要 YUV → RGB 转换)
- ❌ 音频播放 (需要集成 SDL2)
- ❌ 流分析功能
- ❌ 人脸检测
- ❌ 直方图分析
- ❌ 硬件解码

## 📊 技术实现

### UI 组件层级

```
MainWindow (QMainWindow)
├── centralWidget (QWidget)
│   ├── video_label (QLabel) - 视频显示
│   ├── control_group (QGroupBox) - 控制面板
│   │   ├── play_button (QPushButton)
│   │   ├── pause_button (QPushButton)
│   │   ├── stop_button (QPushButton)
│   │   ├── seek_slider (QSlider)
│   │   └── time_label (QLabel)
│   ├── tab_widget (QTabWidget)
│   │   ├── info_tab (流信息)
│   │   │   └── info_text (QTextEdit)
│   │   └── analysis_tab (分析结果)
│   └── info_label (QLabel) - 底部状态
├── menuBar (QMenuBar)
├── toolBar (QToolBar)
└── statusBar (QStatusBar)
```

### 信号槽连接

```cpp
// 播放器 → UI
MediaPlayer::StateChanged → MainWindow::OnStateChanged
MediaPlayer::FrameReady → MainWindow::OnFrameReady
MediaPlayer::PositionChanged → MainWindow::OnPositionChanged
MediaPlayer::Error → MainWindow::OnError

// UI → 播放器
QPushButton::clicked → MediaPlayer::Play/Pause/Stop
QSlider::valueChanged → MediaPlayer::Seek
```

## 🔧 故障排除

### 如果窗口没有出现

1. **检查进程**:
   ```bash
   ps aux | grep VideoEye
   ```

2. **检查错误输出**:
   ```bash
   cd /home/hxk/project/VideoEye/VideoEye/build
   ./bin/VideoEye
   ```

3. **设置显示变量**:
   ```bash
   export DISPLAY=:0
   export QT_QPA_PLATFORM=xcb
   ./bin/VideoEye
   ```

### 如果界面显示异常

- **最小尺寸**: 窗口不能小于 1200x800
- **字体问题**: 检查系统字体配置
- **图标缺失**: 确保 Qt 图标主题已安装
